# Soundcore Desktop Config

### Soundcore API
Currently allows you to control:
- ANC Modes: ANC (Transport), Transparency, and Normal.

The Soundcore R50i NC is the only device I support.

### Requirements
- Python 3.9.x
- tkinter
- pybluez2

### How to Use
- Download as zip/Clone this repo
- Open terminal
- cd to the folder (important to detect the images)
- type `py SoundcoreDesktop.py` enter
- press things

### Tips
If the ANC Modes button didnt do anything, try pressing the connect button <u>just 1 time</u>, then try switching the ANC Modes again
